# Troubleshooting

- **I do not see styles or cards**: Make sure you opened `web/index.html`, not just the repo folder.
- **My queue disappeared**: Browser local storage may have been cleared. Rebuild the queue or restore it from an exported JSON backup if you have one.
- **I expected real scheduling or OAuth**: The local planner is real now, but the live publishing integrations are still a later phase.
- **The page looks off on mobile**: Refresh once and retry. If it still looks wrong, note the device size for a later responsive pass.
