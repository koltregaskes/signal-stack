# Troubleshooting

- **I do not see styles or cards**: Make sure you opened `web/index.html`, not just the repo folder.
- **The preview looks static**: That is expected. This repo currently ships a local mockup, not the backend product.
- **I expected real scheduling or OAuth**: Those pieces are still planned and documented, but not wired yet.
- **The page looks off on mobile**: Refresh once and retry. If it still looks wrong, note the device size for a later responsive pass.
