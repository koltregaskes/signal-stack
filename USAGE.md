# Usage

1. Open the static preview in your browser.
2. Type a sample post in the composer area.
3. Use the quote toggle to see where quoted X links will be handled.
4. Review the platform chips, groups, and schedule fields to understand the intended posting flow.
5. Scan the weekly calendar and validation panel to see how scheduled posts and rule checks will surface.

## Current scope
- This is a UI preview, not a live scheduler yet.
- Buttons and controls show the intended workflow rather than executing real API actions.
