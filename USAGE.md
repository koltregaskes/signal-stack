# Usage

1. Open the app and create a post in the composer.
2. Pick your targets across Instagram, TikTok, YouTube, and Shorts.
3. Set media type, ratio, duration, and schedule time.
4. Save the item as a draft or scheduled post.
5. Load queue items back into the composer to edit or duplicate them.
6. Watch the validation rail for format and platform-fit warnings.
7. Export the queue as JSON when you want a backup or handoff.

## Current scope
- This is a real local scheduler, but not a live publisher yet.
- It handles local planning, storage, validation, and queue shaping.
- It does not yet authenticate with the social platforms or upload media.
