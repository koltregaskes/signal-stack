Signal Stack Privacy Policy

This document is a production-ready starter for a personal or small-team deployment of Signal Stack.

Update the owner name, contact email, hosting details, and retention periods before using this page in a formal platform review.

What Signal Stack Stores

Signal Stack stores queue items, account labels, publishing schedules, upload metadata, local media files, alerts, and activity logs so the app can plan and deliver creator posts.

Connected Accounts

When a platform account is connected through OAuth, Signal Stack may store access tokens, refresh tokens, account identifiers, granted scopes, and publishing capability metadata. In this build, token material is intended to be encrypted at rest when a token encryption key is configured.

How Data Is Used

Data is used only to let the account owner schedule posts, validate payloads, connect supported platforms, upload media, and trigger publishing actions through official provider APIs.

Data Sharing

Signal Stack shares data only with the platforms the account owner explicitly connects for publishing workflows, such as YouTube, Instagram, or TikTok.

Retention

Queue data, uploads, and connector metadata remain stored until the app operator deletes them or resets the deployment storage.

Security

This deployment can be protected with an app password, signed sessions, upload size limits, origin checks for mutating requests, and encrypted connector credentials when the recommended environment variables are set.

Your Choices

The app operator can disconnect linked accounts, remove local queue items, delete uploaded media, and rotate application secrets.
